_.o0 Ganymede 0o._
==========================

![alt text](http://i.imgur.com/Dl4CHyT.png "Ganymede")

HTML5/Javascript web browser game. 

# Controls
Collect as many carrots as possible in **3 minutes**. Use the controls to move and repel opponents.

  * **Arrow keys** - Move ship
  * **Z (or Y)** - Shoot
